# Godot Infinite Grid

A Godot 4.7 project containing reusable `InfiniteGrid3D` and `InfiniteGrid2D` scenes and shaders.

## Run

Open the project in Godot and run `res://examples/scene-3d.tscn` or `res://examples/scene-2d.tscn`, or run the project directly from the editor.

## Structure

```text
addons/                 Reusable Godot addon-style components.
  infinite_grid/         Infinite 2D/3D grid scenes, scripts, and shaders.
docs/                   Project documentation.
entities/               Example support scenes and scripts.
  free_camera.*         Editor-style debug camera.
examples/               Runnable examples.
  scene-2d.*            Minimal 2D infinite grid example scene.
  scene-3d.*            Minimal 3D infinite grid example scene.
```
