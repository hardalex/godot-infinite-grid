# Godot Endless Grid

A Godot 4.7 project containing a reusable `EndlessGrid3D` scene and shader.

## Run

Open the project in Godot and run `res://examples/scene-3d.tscn`, or run the project directly from the editor.

## Structure

```text
addons/                 Reusable Godot addon-style components.
  endless_grid/         Endless XZ grid scene, script, and shader.
docs/                   Project documentation.
entities/               Example support scenes and scripts.
  free_camera.*         Editor-style debug camera.
examples/               Runnable examples.
  scene-3d.*            Minimal endless grid example scene.
```
